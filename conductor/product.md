# Initial Concept
A Discord bot that captures system audio or microphone input via FFmpeg and streams it directly to a Discord voice channel. It features an interactive CLI for device selection and uses modern Discord.js voice libraries.

# Product Vision
AudioBot is a specialized Discord utility designed for small groups and friend circles who need a reliable, high-quality way to share audio directly from a Windows host into their voice channels.

# Target Users
- Small groups and friend circles sharing music, gameplay audio, or specialized high-quality audio streams.

# Project Goals
- **Low Latency:** Optimized for real-time audio delivery from Windows hosts.
- **High Fidelity:** Ensuring stable, crystal-clear connections even during long streaming sessions.

# Core Features
- **Interactive Device Selection:** A user-friendly CLI interface to scan and select system audio inputs.
- **Discord Controls:** Real-time `!join` and `!stop` commands for seamless control within the chat.
- **Configurable Quality:** Support for adjustable bitrate and sample rates to suit different network conditions.

# Success Criteria
- **Audio Quality:** Zero-stutter performance without robotic artifacts.
- **Efficiency:** Minimal CPU and Memory footprint on the host system.
- **Stability:** Maintaining high stability during long-running streaming sessions.
